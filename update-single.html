21.0.7.153

+ [Розробник] Додана можливість блокування видаткові операції по рахунку у разі якщо рахунок не взято на облік в ДФС. Зверніть увагу даний механізм актуальний у разі налаштувань, автоматичного обміну з ДФС

* [Розробник] Зміна реєстру власників для процедури передачі на відповідальне зберігання. Для фізичної особи додані дата та міста народження, інформація про документ

* [Розробник] Виправлена помилка при виконанні розпорядження НБУ у разі відсутності номеру рахунку

* [Розробник] Оновлено механізм формування документу обміну з НБУ для операцій поставка проти оплати

+ [Розробник] Для операцій поставка проти оплати в НБУ доданий параметр версія СЕП

+ [Розробник] В банківські реквізити доданий реквізит ID-код клієнта (використовується для операцій поставка проти оплати у депозитаріях НДУ та НБУ (після впровадження СЕП4)) заповнюється відповідно «Система електронних платежів Національного банку України (шифр СЕП-4) Загальні правила реалізації стандарту ISO 20022, спільні для всіх інструментів Частина 2. Ідентифікація»(https://bank.gov.ua/files/ISO20022/Pravula.identufikazii.NBU.ver.2.3_22.02.22.pdf)

+ [Розробник] В розпорядження поставка проти оплати додано додатковий реквізит ID-код клієнта, заповнюється відповідно «Система електронних платежів Національного банку України (шифр СЕП-4) Загальні правила реалізації стандарту ISO 20022, спільні для всіх інструментів Частина 2. Ідентифікація»(https://bank.gov.ua/files/ISO20022/Pravula.identufikazii.NBU.ver.2.3_22.02.22.pdf)

+ [Розробник] Для операцій поставка проти оплати в НБУ доданий новий статус «Очікує оплати»

+ [Розробник] Для операцій НБУ у разі використання серверного варіант підпису додано можливість виконання операцій з журналу розпоряджень без необхідності відкриття розпорядження

* [Розробник] Змінено процедуру обробки операцій погашення та блокування на погашення в депозитарії НБУ у разі використання депозитарною установою додаткових агрегованих рахунків для фізичних осіб, юридичних осіб, нерезидентів

* [Розробник] Оновлений сервіс шлюзу взаємодії з ДІЄю. у тому числі оновлення Список сертифікатів сумісних ЦСК та Параметри взаємодії із сумісними ЦСК

* [Розробник] Оновлення end-point взаємодії з ДІЄю. Оновлення end-point також дозволяє використовувати один end-point для декількох підприємств
